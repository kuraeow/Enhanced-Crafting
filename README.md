# Enhanced Crafting

Standalone adaptation of the popular **“Enhanced Crafting UI”** concept for **WoW Midnight**.

`/ecraft` — open addon settings.

## Requirements

- **Ace3** is required.

## Features

- **Unlock All Reagents**  
  Sets reagent counts to `9999` for preview purposes (including optional reagents), so you can simulate different material quality setups.

- **Inspiration Result Preview**  
  Shows expected result when Inspiration procs, helping estimate possible quality/rank upgrades.

- **Crafter Search & Skill Preview**  
  Lets you find crafters for the selected recipe and preview their expected result with your chosen materials.

- **Personal Order Alert**  
  Displays a configurable reminder when a new Personal Crafting Order is received.

- **Quick Place Order Access**  
  Adds fast access to the Place Order window from the profession UI.

- **Localization**
  - English (`enUS`)
  - Russian (`ruRU`)

## Notes

- This is a standalone addon and does not require WeakAuras.
- Some features rely on Blizzard UI and chat/system message behavior, which can differ slightly by client version/localization.
